# SymGov BTX Integration Handoff

Start with `CODEX_TASK.md` and submit that file to Codex with this folder available in the repository or workspace.

The package contains:

- a Codex-ready integration specification;
- the validated reference extractor, enhanced to resolve the exact `/AP /N` resource;
- the original `doors.zip` and unpacked `Doors.btx` fixtures;
- expected manifest values;
- golden SVG and DXF output;
- a visual preview.

The GitHub connector did not expose the SymGov repository in the session that created this package, so the task deliberately instructs Codex to inspect and follow the repository's existing stack and conventions.
